GitHub update builder

Build update JSON for GitHub.

Upload your APK to GitHub Releases, paste the release asset URL here, and this page creates the JSON that the app reads from GitHub.

No Firebase Storage needed This page does not upload anything. It only computes the APK SHA-256 and creates the update manifest.
App build setting Build the app with --dart-define=UPDATE_MANIFEST_URL=https://raw.githubusercontent.com/OWNER/REPO/main/update.json
